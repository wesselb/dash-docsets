Defined in python/keras/mixed_precision/experimental/policy.py.

The global policy is the default policy used for layers, if no policy is passed to the layer constructor. When TensorFlow starts, the global policy is set to an "infer" policy, and can be changed with set_policy.

Returns:

The global Policy.
