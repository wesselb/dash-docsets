tf.is_variable_initialized

is_variable_initialized(variable)

Defined in tensorflow/python/ops/variables.py.

See the guide: Variables > Variable helper functions

Tests if a variable has been initialized.

Args:

  • variable: A Variable.

Returns:

Returns a scalar boolean Tensor, True if the variable has been initialized, False otherwise.

NOTE The output of this function should be used. If it is not, a warning will be logged. To mark the output as used, call its .mark_used() method.
