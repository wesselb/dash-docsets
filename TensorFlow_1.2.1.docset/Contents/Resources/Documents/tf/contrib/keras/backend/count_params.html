tf.contrib.keras.backend.count_params

tf.contrib.keras.backend.count_params

count_params(x)

Defined in tensorflow/contrib/keras/python/keras/backend.py.

Returns the number of scalars in a Keras variable.

Arguments:

x: Keras variable.

Returns:

Integer, the number of scalars in `x`.

Example:

    >>> kvar = K.zeros((2,3))
    >>> K.count_params(kvar)
    6
    >>> K.eval(kvar)
    array([[ 0.,  0.,  0.],
           [ 0.,  0.,  0.]], dtype=float32)
