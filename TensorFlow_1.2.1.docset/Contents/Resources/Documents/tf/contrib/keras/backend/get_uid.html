tf.contrib.keras.backend.get_uid

tf.contrib.keras.backend.get_uid

get_uid(prefix='')

Defined in tensorflow/contrib/keras/python/keras/backend.py.

Associates a string prefix with an integer counter in a TensorFlow graph.

Arguments:

  • prefix: String prefix to index.

Returns:

Unique integer ID.

Example:

  >>> get_uid('dense')
  1
  >>> get_uid('dense')
  2
