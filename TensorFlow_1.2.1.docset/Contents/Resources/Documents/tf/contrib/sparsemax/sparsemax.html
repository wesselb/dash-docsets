Defined in tensorflow/contrib/sparsemax/python/ops/sparsemax.py.

Computes sparsemax activations 1.

For each batch i and class j we have sparsemax[i, j] = max(logits[i, j] - tau(logits[i, :]), 0)

Args:

  • logits: A Tensor. Must be one of the following types: half, float32, float64.
  • name: A name for the operation (optional).

Returns:

A Tensor. Has the same type as logits.
