bucketized_column(

)

Defined in tensorflow/contrib/layers/python/layers/feature_column.py.

See the guide: Layers (contrib) > Feature columns

Creates a _BucketizedColumn for discretizing dense input.

Args:

  • source_column: A _RealValuedColumn defining dense column.
  • boundaries: A list or tuple of floats specifying the boundaries. It has to be sorted.

Returns:

A _BucketizedColumn.

Raises:

  • ValueError: if 'boundaries' is empty or not sorted.
