tf.gfile.Copy

tf.gfile.Copy

Copy(
    oldpath,
    newpath,
    overwrite=False
)

Defined in tensorflow/python/lib/io/file_io.py.

Copies data from oldpath to newpath.

Args:

  • oldpath: string, name of the file who's contents need to be copied
  • newpath: string, name of the file to which to copy to
  • overwrite: boolean, if false its an error for newpath to be occupied by an existing file.

Raises:

errors.OpError: If the operation fails.
